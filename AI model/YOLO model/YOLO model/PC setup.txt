Windows 11 এ YOLO setup + check 👇



STEP 1: Python install (Windows 11)
Download: 👉 https://www.python.org/downloads/windows/
Install করার সময় IMPORTANT: ✔ “Add Python to PATH” ✔️ (এইটা অবশ্যই tick দাও)


STEP 2: Test Python install
CMD (Command Prompt) খুলে লিখো:
python --version
Output আসবে → দেখতে পাবে Python version


STEP 3: YOLO + Libraries install
CMD এ এইটা paste করো: pip install ultralytics opencv-python requests
